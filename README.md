# 11022233土木四乙張翔禎 公車查詢系統

## 介紹
本專案為公車路線查詢加上起終點路線最快路線查詢系統，可以顯示輸入的公車代碼去返程路線，以及輸入起終點站可以在地圖顯示路線以及起終點位置與公車目前位置

## 安裝方式

1. 安裝 Python 3.7 以上版本
2. 安裝必要套件

   pip install -r requirements.txt
   python -m playwright install
    pip install selenium
